HTML

HTML (HyperText Markup Language) é uma linguagem padrão utilizada para estruturar e apresentar o conteúdo de páginas da web. Ele foi desenvolvido inicialmente por Tim Berners-Lee em 1991 e é um dos pilares da web, junto com CSS e JavaScript.

CSS

O CSS (Cascading Style Sheets), ou Folhas de Estilo em Cascata, é uma linguagem de estilo utilizada para definir a aparência e o layout de páginas web. Ela é frequentemente usada em conjunto com o HTML para criar páginas web visualmente atraentes e funcionais. O CSS permite que você controle uma variedade de estilos, como cores, fontes, espaçamento, tamanhos e posicionamento de elementos na página.
